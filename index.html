安全说明

  • 点击「参与并支付」将发起链上交易；请以钱包弹窗中的收款地址与金额为准。
  • 合约限制：最多 1000 个成功参与地址；msg.sender 不得为合约地址（code.length != 0 即拒绝）。
  • 单笔金额由合约常量 minWei / maxWei 固定（本计划当前为小额区间），部署不可改；constructor 仅设 treasury、admin。
  • admin 可 sweep* 转走合约内误转资产；请使用可信多签。
  • 链上只读与交易均通过钱包扩展所配置的 RPC；前端依赖的 ethers 与页面同源分发，不依赖第三方脚本 CDN。
